# Map Trip - Roadtrip Planner

An interactive roadtrip planner built with Leaflet and Vite. Plan routes between stops on a map, add intermediate destinations, and calculate distances.

## Features

- Interactive map powered by Leaflet
- Default route: Stockholm to Paris and back
- Add custom intermediate stops
- Toggle return trip
- Route distance calculation

## Getting Started

### Prerequisites

- Node.js (v20 or later recommended)

### Install

```bash
npm install
```

### Development

```bash
npm run dev
```

Opens a local dev server at `http://localhost:5173/map-trip/`.

### Build

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```

## Deployment

The project auto-deploys to GitHub Pages on push to `main` via GitHub Actions.

## Tech Stack

- [Leaflet](https://leafletjs.com/) - interactive maps
- [Vite](https://vite.dev/) - build tool and dev server
